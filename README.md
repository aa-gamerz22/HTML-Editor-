## ‚ú® Key Features

AH IDE is packed with powerful features to enhance your coding workflow:

* **‚ö° Dual Editor Engine:**
    * `Professional Monaco`: Leverage the power of Microsoft's Monaco Editor (the core of VS Code) for rich syntax highlighting, code completion, and line numbers.
    * `Native Simple`: A lightweight, fast-selecting mode for quick edits and basic text manipulation.
* **üíé Elite Library Hub:**
    * Access to `25+ popular JavaScript & CSS libraries` (e.g., Three.js, GSAP, Tailwind CSS, Matter.js, Axios, React, Lottie, SweetAlert2).
    * `One-click installation`: Easily add libraries directly into your project folder.
    * `Installing Animations`: Smooth visual feedback during library downloads.
* **üõ†Ô∏è Control Center (IDE Settings):**
    * `Live Font Scaling`: Adjust editor font size instantly (10px to 40px).
    * `Word Wrap Toggle`: Switch between `Fit Content` (word wrap ON) and `Fill Content` (word wrap OFF for side-scrolling).
    * `Elite Themes`: Choose from 4 stunning UI themes: `Deep Space`, `Pure OLED`, `Cyber Neon`, and `Gold Luxury`.

## üöÄ Getting Started

To run AH IDE locally:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/ah-ide.git](https://github.com/your-username/ah-ide.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd ah-ide
    ```
3.  **Open `index.html`:** Simply open the `index.html` file in your preferred web browser.

**Note:** For full functionality (like file system access), it's recommended to run AH IDE on a modern browser that supports the File System Access API (e.g., Chrome, Edge).
